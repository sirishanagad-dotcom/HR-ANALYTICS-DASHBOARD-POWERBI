📊 HR Analytics Dashboard (Power BI)

This project showcases an interactive HR Analytics Dashboard built using Power BI. It provides insights into employee data, including attrition trends, salary distribution, performance ratings, and workforce demographics.

🔍 Key Features
Employee distribution by department and gender
Attrition analysis by department
Salary insights across roles
Performance rating distribution
Interactive slicers (Department, Job Role, Gender, Year)
📸 Dashboard Preview




🛠️ Tools Used
Power BI
Data Modeling
DAX (Basic Measures)
🎯 Purpose

The dashboard helps organizations make data-driven HR decisions and understand workforce patterns effectively.
